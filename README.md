# webs-vty-archive

这是一个用于归档和发布多个独立 HTML 页面的仓库。仓库中的页面内容彼此独立，不针对任何特定域名或具体网站。本仓库旨在以静态形式保存和展示一些 Web 页面资源，供浏览、参考或长期存档使用。

## 项目简介

webs-vty-archive 是一个轻量级的静态页面存档项目。所有页面以 HTML 文件形式存放，不依赖后端服务或数据库。每个页面可以单独访问，适合作为历史资料、示例页面或小型工具页面的集合。

## 目录结构

```
.
├── index.html          # 仓库首页 / 页面列表
├── pages/              # 存放归档页面的目录
│   ├── example-01.html
│   ├── example-02.html
│   └── ...
└── assets/             # 公共资源（CSS、JS、图片等）
    ├── style.css
    └── ...
```

- `index.html`：仓库的入口页面，通常包含页面列表或导航。
- `pages/`：所有归档的独立 HTML 页面存放于此。
- `assets/`：页面共享的样式、脚本、图片等资源文件。

## 页面归档说明

- 每个页面均为独立的 HTML 文件，可直接在浏览器中打开。
- 页面之间没有强依赖关系，可单独维护或更新。
- 归档内容可能包含示例数据、演示界面或历史版本快照。
- 所有页面均不包含外部跟踪、分析或商业推广代码。
